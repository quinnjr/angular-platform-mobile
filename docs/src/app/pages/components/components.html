Components

Angular Platform Mobile provides a comprehensive set of native UI components that map directly to platform-specific views. Each component is designed to look and feel native on both iOS and Android.

Core Components

These components form the foundation of your mobile UI. They correspond directly to native views and provide full access to platform capabilities.

@for (component of components; track component.path) {
{{ component.icon }}

{{ component.name }}

{{ component.description }}

@for (platform of component.platforms; track platform) { {{ platform }} }
}

Component Naming

All components use the mobile- prefix in their selectors: 

<!-- View component -->
<mobile-view [style]="styles.container">

  <!-- Text component -->
  <mobile-text [style]="styles.title">Hello World</mobile-text>

  <!-- Button component -->
  <mobile-button (press)="onPress()">Click Me</mobile-button>

</mobile-view>

Common Props

All components share these common properties:

Prop Type Description
style ViewStyle | TextStyle | ImageStyle Styling for the component
testID string Identifier for testing frameworks
accessible boolean Whether the component is accessible
accessibilityLabel string Label read by screen readers
accessibilityHint string Hint for what happens on interaction

Importing Components

Import components from the main package:

import {
  ViewComponent,
  TextComponent,
  ImageComponent,
  ButtonComponent,
  TextInputComponent,
  ScrollViewComponent,
  FlatListComponent,
  ModalComponent,
  WebViewComponent,
  SwitchComponent,
  SliderComponent,
  StatusBarComponent,
  ActivityIndicatorComponent,
} from '@pegasusheavy/angular-platform-mobile';

Or use the module that includes all components:

import { MobileComponentsModule } from '@pegasusheavy/angular-platform-mobile';

@Component({
  imports: [MobileComponentsModule],
  // ...
})
