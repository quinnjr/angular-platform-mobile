Getting Started

Get up and running with Angular Platform Mobile in just a few minutes. This guide covers installation, project setup, and building your first native app.

Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js 18.0.0 or later
  • pnpm (recommended) or npm
  • Angular CLI 18.0.0 or later
  • For Android: Android Studio with SDK 24+
  • For iOS: Xcode 15+ (macOS only)

Installation

Create a New Project

The fastest way to get started is using our CLI:

# Create a new project
npx ng-mobile init my-app

# Navigate to the project
cd my-app

# Install dependencies
pnpm install

Add to Existing Angular Project

You can also add Angular Platform Mobile to an existing Angular project:

# Install the package
pnpm add @pegasusheavy/angular-platform-mobile

# Initialize mobile platform
npx ng-mobile init --existing

Your First App

Bootstrap the Application

Update your main.ts to use the mobile bootstrap:

import { bootstrapMobile } from '@pegasusheavy/angular-platform-mobile';
import { AppComponent } from './app/app.component';

bootstrapMobile(AppComponent, {
  config: {
    debug: true,
    hotReload: true,
  },
}).catch(err => console.error(err));

Create Your First Component

Replace your app component with native mobile components:

import { Component } from '@angular/core';
import {
  ViewComponent,
  TextComponent,
  ButtonComponent,
  StyleSheet,
} from '@pegasusheavy/angular-platform-mobile';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [ViewComponent, TextComponent, ButtonComponent],
  template: \`
    <mobile-view [style]="styles.container">
      <mobile-text [style]="styles.title">
        Welcome to Angular Mobile!
      </mobile-text>

      <mobile-text [style]="styles.subtitle">
        Build native apps with Angular
      </mobile-text>

      <mobile-button
        [style]="styles.button"
        (press)="onButtonPress()">
        Get Started
      </mobile-button>
    </mobile-view>
  \`,
})
export class AppComponent {
  styles = StyleSheet.create({
    container: {
      flex: 1,
      justifyContent: 'center',
      alignItems: 'center',
      backgroundColor: '#ffffff',
      padding: 20,
    },
    title: {
      fontSize: 28,
      fontWeight: 'bold',
      color: '#1a1a1a',
      marginBottom: 8,
    },
    subtitle: {
      fontSize: 16,
      color: '#666666',
      marginBottom: 32,
    },
    button: {
      backgroundColor: '#dd0031',
      paddingVertical: 14,
      paddingHorizontal: 32,
      borderRadius: 8,
    },
  });

  onButtonPress(): void {
    console.log('Button pressed!');
  }
}

Running Your App

Development Mode

Start the development server with hot reload:

# Start the dev server
pnpm start

# Or use the CLI
npx ng-mobile start

Run on Android

# Run on connected Android device or emulator
npx ng-mobile run android

# Build a release APK
npx ng-mobile build android --release

Run on iOS (macOS only)

# Run on iOS Simulator
npx ng-mobile run ios

# Run on specific simulator
npx ng-mobile run ios --simulator="iPhone 15 Pro"

# Build for App Store
npx ng-mobile build ios --release

Platform-Specific Code

Use Platform.select() to write platform-specific logic:

import { Platform } from '@pegasusheavy/angular-platform-mobile';

const styles = StyleSheet.create({
  container: {
    padding: Platform.select({
      ios: 20,
      android: 16,
    }),
    ...Platform.select({
      ios: {
        shadowColor: '#000',
        shadowOffset: { width: 0, height: 2 },
        shadowOpacity: 0.1,
        shadowRadius: 4,
      },
      android: {
        elevation: 4,
      },
    }),
  },
});
